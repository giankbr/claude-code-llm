Get Started in 5 Minutes

1. Clone & Install

git clone <repo-url>
cd agent-ai-llm
bun install

2. Choose Your Provider

For Anthropic: Get API key from console.anthropic.com

For Ollama: Install from ollama.ai and run ollama run mistral

3. Configure .env

PROVIDER=ollama
OLLAMA_MODEL=mistral
# or use PROVIDER=anthropic with ANTHROPIC_API_KEY

4. Launch

bun run index.ts

5. Start Coding

> read package.json
> write a function that validates emails
> run npm test

Example: Writing a Function

Human

create a function that checks if an email is valid

Assistant

I'll create an email validation function for you:

function isValidEmail(email) {
  const regex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
  return regex.test(email);
}

// Test it:
console.log(isValidEmail("test@example.com")); // true
console.log(isValidEmail("invalid.email"));     // false

This regex checks for the basic email format: something@something.something

Human

write it to a file called email-validator.js

Assistant

Done! I've created the file email-validator.js with the validation function.
